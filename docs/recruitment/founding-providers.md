## Recruitment Channels

1. **Personal network** — warm intros to rabbis you know
2. **Seminary alumni groups** — JTS, HUC, YU, RRC, Aleph
3. **Neshama** — Association of Jewish Chaplains directory
4. **LinkedIn** — search "rabbi pastoral counseling" + "spiritual direction"
5. **Existing platforms** — scholars on Moreh Derech, independent pastoral counselors

## Vetting Checklist (Per Provider)

- Application received via `/apply`
- Ordination / credentials verified
- 1–2 reference calls completed
- Background check ordered
- Provider agreement signed
- Boundary training completed
- Profile approved in admin dashboard
- Test booking flow verified

## Compensation Summary

- Provider sets rate (suggested: $75–150 / 45–60 min)
- Platform fee: 18%
- Payouts via Stripe Connect (when configured)
